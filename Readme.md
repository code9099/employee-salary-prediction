A machine learning web application that predicts employee salaries based on various factors like experience, education, job title, and more.

## Features
- Interactive salary prediction interface
- Multiple ML algorithms comparison (Random Forest, Decision Tree, Linear Regression)
- Data visualization and analysis
- Real-time predictions with Streamlit

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run salary_prediction_app.py`
3. Open your browser and go to the provided URL

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib/Seaborn
